# **The Ontology of Self-Organizing Forms**

### *A Unified Variational Field Theory of Structure, Meaning, and Identity*

This repository contains an integrated set of documents explaining, contextualizing, and formalizing the unified framework developed in **The Ontology of Self-Organizing Forms**, a monograph introducing the **Relativistic Scalar–Vector Plenum (RSVP)** and its deep equivalence with **Spherepop semantics**, the **Yarncrawler inference operator**, and the **Free Energy Principle (FEP)**.

The core claim of the framework is that physics, semantics, and inference are not separate explanatory layers, but **different coordinate charts** on a single underlying geometric and thermodynamic system. When expressed through renormalization, this system naturally gives rise to **Markov blankets**, **self-organizing forms**, **agents**, **meaning**, and **consciousness**.

This repository offers both high-level conceptual introductions and rigorous technical documents. The files are designed to interlock: readers can begin at any level of familiarity and gradually move toward the full mathematical theory.
